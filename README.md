### 万字文档+花店管理系统+SSM+Vue
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、Vue2、JavaScript、CSS、HTML

系统功能模块：
本系统包含管理员、用户2个角色。
管理员角色：花材选择管理、用户管理、鲜花管理、鲜花出入库管理、鲜花订单管理等。
用户角色：联系客服、查看鲜花、收藏鲜花、购买鲜花、评论鲜花、支付订单、管理个人订单等。
